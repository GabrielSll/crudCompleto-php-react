# CRUD COMPLETO

### Como inicar o projeto: 

- Entre no diretório do projeto: cd newm-selecao
- Instale todas as dependências

npm:

```sh
npm install
```

Rode a aplicação em seu servidor local: 

```sh
npm start
```
